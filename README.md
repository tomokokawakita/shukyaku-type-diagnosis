# あなたにぴったりの集客タイプ診断

女性起業家・個人事業主・講師・フリーランス向けの、登録不要で使えるWeb診断です。

## 使い方

`index.html` をブラウザで開くだけで利用できます。サーバーやデータベースは不要です。

## 診断内容

- 16問・5段階回答
- 6つの集客タイプからメインタイプを診断
- 相性のよいサブタイプを2つ表示
- 向いている集客方法、おすすめの発信、最初の3ステップを提案
- 結果のシェア機能

診断結果や回答内容は保存・送信されず、すべてブラウザ内だけで処理されます。

## 公開

GitHub Pagesで静的サイトとして公開できます。ビルド処理は不要です。
